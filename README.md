Pokedex Diary

A multi-page Pokémon application built with Vite, vanilla JavaScript, and TailwindCSS.  
The project was refactored from a legacy JavaScript codebase into a modular ES module architecture.

---

Tech Stack

Vite
Vanilla JavaScript (ES Modules)
TailwindCSS
PokéAPI
LocalStorage

---

Setup

npm install
npm run dev
npm run build

---

Features

Pokémon data from PokéAPI
Pokémon cards with stats
Multi-page application (MPA)
Favorites system (LocalStorage)
Modular ES module structure
Clean refactored architecture
